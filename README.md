# Kullanıcı Giriş ve Yönetim Paneli - Windows Forms (C#)

Bu projede kullanıcı kayıt ve giriş sistemi, “şifremi unuttum” özelliği, SQL Server tabanlı admin paneli bulunmaktadır.  
Kullanıcılara doğrulama kodu e-posta ile gönderilir. Veriler ListView ve DataGridView üzerinde listelenip sıralanabilir, eklenebilir, silinebilir ve güncellenebilir.
